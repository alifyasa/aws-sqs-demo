# Demo Amazon SQS

Dibuat untuk demonstrasi AWS SQS di mata kuliah IF4031 Pengembangan Aplikasi Terdistribusi.

## Cara Penggunaan

1.  `yarn`
2.  Set _credentials_ in `.env`
3.  `yarn start`
